# React Chat App

this app is made using react,styled component,socket.io, You can send and receive messages from the other party in real time.


## How to Run a Project

-after cloning the project follow these steps
* `cd web`
* `npm install`
* `npm start`
* `cd ..`
* `cd server`
* `npm install`
* `npm start`
